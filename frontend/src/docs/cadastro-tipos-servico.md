# Tipos de Serviço

## Objetivo
Manter a lista de **tipos de serviço** oferecidos (ex.: Psicoterapia, Avaliação,
Reabilitação). É uma **lista de apoio**.

## Como acessar
Menu **Cadastro → Tipos de Serviço** (`/cadastro/tipos-servico`).

## Campos do formulário
| Campo | Obrigatório | Descrição |
|-------|:----------:|-----------|
| Nome | Sim | Nome do tipo de serviço. |

## Onde esta lista é usada
- Na tela **Serviços do Paciente** (`/cadastro/paciente-servico`), no campo
  **Tipo de Serviço**.

## Regras e observações
- Tela de manutenção simples: adicionar, editar e excluir itens.
