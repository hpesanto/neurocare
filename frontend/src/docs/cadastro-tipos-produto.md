# Tipos de Produto

## Objetivo
Manter a lista de **categorias de produto** (ex.: Material, Livro, Kit). É uma
**lista de apoio** para classificar os produtos.

## Como acessar
Menu **Cadastro → Tipos de Produto** (`/cadastro/tipos-produto`).

## Campos do formulário
| Campo | Obrigatório | Descrição |
|-------|:----------:|-----------|
| Nome | Sim | Nome do tipo de produto. |

## Onde esta lista é usada
- No cadastro de **Produtos**, para classificar cada produto por tipo.

## Regras e observações
- Tela de manutenção simples: adicionar, editar e excluir itens.
