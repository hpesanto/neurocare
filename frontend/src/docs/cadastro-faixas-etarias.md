# Faixas Etárias

## Objetivo
Manter a lista de **faixas etárias** (ex.: Criança, Adolescente, Adulto, Idoso).
É uma **lista de apoio** usada para classificar pacientes.

## Como acessar
Menu **Cadastro → Faixas Etárias** (`/cadastro/faixas-etarias`).

## Campos do formulário
| Campo | Obrigatório | Descrição |
|-------|:----------:|-----------|
| Nome | Sim | Nome da faixa etária. |

## Onde esta lista é usada
- No cadastro de **Pacientes**, no campo **Faixa Etária**.

## Regras e observações
- Tela de manutenção simples: adicionar, editar e excluir itens.
