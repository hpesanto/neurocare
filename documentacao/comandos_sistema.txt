### criando super usuário para acesso ao painel
Criar superusuário (administrador do Django): python manage.py createsuperuser (use para acessar o painel de administração futuramente).

Teste Inicial:
Rodar o servidor de desenvolvimento: python neurocare_system/manage.py runserver
Acesse http://127.0.0.1:8000/admin/ no seu navegador. Você deve ver a tela de login. Faça login com o superusuário.
